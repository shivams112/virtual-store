Context

You have learnt how to publish a message to a Facebook page using the python requests module. It is now time to run the full QEats backend server. The code you wrote in the previous module will be used here to publish reviews coming from the QEats Android app.

To write a review for an order, the QEats Android app will prompt users to first take a picture (or select one from their image gallery). They can then add a description and some #hashtags.

What you will do in this module

In the module, you will install the QEats app on your phone and post a review to the QEats Facebook page using it.

To post reviews to the QEats Facebook page, you have to start the QEats backend server and configure the Android app to connect to it. You can then post QEats reviews from your phone to the QEats Facebook page.

Note

  1. You dont need a Facebook account to complete this module.

  2. In the full QEats Micro-Experience, you will implement all parts of the app shown in the images above from scratch.

Primary Goals

  1. Install the QEats app.

  2. Run the QEats backend server. Watch this video to understand how.

  3. Use share_review.py to publish a photo message to the QEats Review Facebook page from the command line.

  4. Use the QEats app to share a review directly to the QEats eview Facebook page.
